This Application is for educational purposes only and is created for Master Thesis. Application is inspired by British Broadcasting Corporations(BBC) Quiz show Connecting Wall!

Registered users can play quizzes created by other users, as well as create and modify their own quizzes.


Rules of the game :

There are 16 terms randomly sorted. Each term belongs to one of four group of terms (every group consists of 4 terms). First objective of the game is to determine which terms belong to same group and players task is to group them by selecting (marking) them. If 4 selected terms belong to the same group, user gets 1 point. After connecting two groups of terms, user gets 3 lives which represent number of tries to group one more set of terms. If player manages to pair the terms for 3 groups, he automatically gets 4 points. Player needs to group the terms in three minutes. After successful or unsuccessful term grouping, player can win more points by guessing the connection between terms.
